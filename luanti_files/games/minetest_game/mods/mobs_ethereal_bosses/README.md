## 👾 MOBS : 👾

#### " PIXIES "
- Health : 80
- Damage : 2
- Drops : No
- Spawn : Underground caves
- Follow : Green, red or blue flowers (pixie color equivalent)

#### " FUNGS  "
- Health : 80
- Damage : 5
- Drops : mushroom brown
- Spawn : Prairie, biome
- curiosity: if you give him orange, he might jump for joy..
- Follow = Strawberry

#### ==================================================================================================

- Drops : Nature Roots,Stick
- Spawn : Mushroom biome

#### " DEPTHS EYE "
- Heath : 20
- Damage : 3
- Drops : EYE
